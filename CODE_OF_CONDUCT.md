# 贡献者公约行为准则

## 我们的承诺

为营造开放、友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、人种、宗教信仰或性取向如何，参与本项目及社区对每个人都是免于骚扰的体验。

我们承诺以有助于建立开放、友善、多元、包容、健康社区的方式行事与互动。

## 我们的准则

有助于营造积极环境的行为包括：

- 对他人展现同理心与善意
- 尊重不同的意见、观点与经历
- 给出并优雅地接受建设性反馈
- 对受我们行为影响的人承担责任、致歉并从中学习
- 着眼于对整个社区而非个人最有利的事

不可接受的行为包括：

- 使用带有性意味的言语或图像，以及任何形式的性关注或挑逗
- 钓鱼、侮辱或贬低性评论，人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可，公开他人的私人信息（如住址或电子邮箱）
- 在职业场合中可被合理认定为不当的其它行为

## 执行职责

社区维护者有责任阐明并执行可接受行为的准则，并对其认定为不当、威胁、冒犯或有害的行为采取适当且公正的纠正措施。

维护者有权移除、编辑或拒绝与本准则不符的评论、提交、代码、wiki 编辑、issue 等贡献，并在适当时说明理由。

## 适用范围

本准则适用于所有社区场合，也适用于个人在公开场合代表社区之时，例如使用官方邮箱、通过官方社交账号发布，或作为指定代表参与线上或线下活动。

## 执行

如遇辱骂、骚扰或其它不可接受的行为，可通过 **yong001124@gmail.com** 向负责执行的维护者报告。所有投诉都会被及时、公正地审查与处理。

维护者有义务尊重报告者的隐私与安全。

## 执行准则

维护者将依据以下《社区影响指南》判定违反本准则行为的后果：

### 1. 纠正
**社区影响**：使用不当言语或被认为不专业、不受欢迎的行为。
**后果**：私下书面警告，说明违规性质并解释为何不当；可能要求公开道歉。

### 2. 警告
**社区影响**：单次或一系列违规。
**后果**：警告并说明持续行为的后果。在指定时间内，不得与相关人员（含主动联系准则执行者）互动。违反将可能导致临时或永久封禁。

### 3. 临时封禁
**社区影响**：严重违规，包括持续的不当行为。
**后果**：在指定时间内禁止与社区进行任何形式的互动或公开沟通。期间不得与相关人员互动，违反将可能导致永久封禁。

### 4. 永久封禁
**社区影响**：表现出违规模式，包括持续不当行为、骚扰个人或攻击／贬低某类人群。
**后果**：永久禁止在社区内进行任何形式的公开互动。
